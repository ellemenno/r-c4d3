Classpublic class JoyBallEvent
InheritanceJoyBallEvent flash.events.Event

A JOY_BALL_MOTION event occurs when a user moves a trackball on the joystick. The field 'which' is the index of the joystick that reported the event and 'ball' is the index of the trackball. Trackballs only return relative motion--the change in position on the ball since it was last polled (last cycle of the event loop) and it is stored in xrel and yrel.
