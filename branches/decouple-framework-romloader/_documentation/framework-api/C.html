out()method 
public static function out(origin:*, value:String, force:Boolean = false, fullPath:Boolean = false):void

Sends message to current logger, with timestamp and calling class prefixed.

Parameters

origin:* — Calling class, typically this, e.g. C.out(this, "hello"). If calling from within a static method, pass the class directly, e.g. C.out(MyStaticClass, "hello from a static method").
 
value:String — Message to trace
 
force:Boolean (default = false) — When true, override enabledness, e.g. for critical error
 
fullPath:Boolean (default = false) — When true, prefix with entire classpath instead of just class name

Examples
Usage: 
C.enable(); // just once
C.out(this, "bar bat baz");
Usage (forced trace): 
C.out(this, "foo happened!", true); // temporary override of enabled
