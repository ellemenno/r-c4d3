Classpublic class RC4D3RomLoader
InheritanceRC4D3RomLoader RomLoader flash.display.Sprite

Loads a valid IGameRom SWF and provides it access to an RC4D3 game controls proxy and a remote high scores proxy. Notes:
  • romloader-config.xml must declare the custom property scoreServer, defining the score server url
