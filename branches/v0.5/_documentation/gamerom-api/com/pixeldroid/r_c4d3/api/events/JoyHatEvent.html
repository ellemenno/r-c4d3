Classpublic class JoyHatEvent
InheritanceJoyHatEvent flash.events.Event

A JOY_HAT_MOTION event occurs when ever a user moves a 4-way hat on the joystick.

The field 'which' is the index of the joystick that reported the event and 'hat' is the index of the hat. 'value' is the current position of the hat encoded as a bitfield.

Use the boolean properties of the event to inquire about various directionalities, or centering , or write your conditionals testing the event value against the predefinied direction constants.
