Classpublic class JoyAxisEvent
InheritanceJoyAxisEvent flash.events.Event

A JOY_AXIS_MOTION event occurs whenever a user moves a joystick along one of its axes. The field 'which' is the index of the joystick that reported the event and 'axis' is the index of the axis. 'value' is the current position of the axis.
