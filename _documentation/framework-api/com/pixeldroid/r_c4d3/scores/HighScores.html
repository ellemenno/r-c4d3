Classpublic class HighScores
ImplementsIGameScoresProxy
SubclassesLocalHighScores, RemoteHighScores

Base class for high score storage and retrieval. Notes:
  • Scores are kept in descending order (highest first)
  • Ties are allowed (same score, different initials), duplicates are not

insert()method 
public function insert(s:Number, i:String):Boolean

Add a score and associated initials string into the high score list, possibly bumping off the lowest score. The new score and initials string will be added in rank order. If adding the new score and initals string creates more than MAX_SCORES entries, then the lowest rank score will be bumped off the list.

Parameters
s:Number — The score to add
 
i:String — The string of initials to add

isEligible()method 
public function isEligible(score:Number):Boolean

A quick rejection test for candidate scores. When false is returned, the score is definitely not eligible, but when true is returned the score may still be ineligible if it is a duplicate of one already stored.

Parameters

Returns
Boolean — true if score is a candidate for inclusion, false if not
