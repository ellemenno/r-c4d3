Classpublic class RemoteGameScoresProxy
InheritanceRemoteGameScoresProxy GameScoresProxy flash.events.EventDispatcher

Extends the (abstract) GameScoresProxy base class to store high scores and initials on a remote server, using a web service that accepts the following parameters:
  • game : String Unique game id
  • format : String "json" or "vrml"
  • data : String Score data--as json string--to store; omission or empty value for data triggers retrieval
and returns the following data:
  • type: String "get" or "set"
  • success : Boolean
  • message : String
  • data : String Score data as json string
Notes:
  • Requires com.adobe.serialization.json.JSON
