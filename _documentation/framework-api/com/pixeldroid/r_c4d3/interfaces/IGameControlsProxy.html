Implementaors provides system specific access to Joystick, Trackball, and Button information.

joystickEventState()method 
public function joystickEventState(state:String, stage:Stage):String

Enable, disable or query current joystick event polling state.

Parameters
state:String — A JoyEventStateEnum value (QUERY, ENABLE or IGNORE)
 
stage:Stage — A reference to the root of the display list

Returns
String — If state is QUERY then the current state is returned, otherwise the new processing state is returned.

Note: In addition to setting the JoystickEventState to ENABLE, Event listeners must be added in order to receive event notifications.

joystickGetAxis()method 
public function joystickGetAxis(joystick:IJoystick, axis:int):int

Get the current state of an axis control on a joystick.

The axis indices start at index 0. The X axis is index 0. The Y axis is index 1.

Parameters
joystick:IJoystick — A reference to the IJoystick to get the axis value from
 
axis:int — A zero-based index selecting the axis to get (0 for X, 1 for Y)

Returns
int — A value ranging from -32,768 to 32,767.
joystickGetBall()method 
public function joystickGetBall(joystick:IJoystick, ball:int, displacement:Point):Boolean

Get the ball axis change since the last poll.

Trackballs only return relative motion since the last call to joystickGetBall, these motion deltas are placed into the displacement parameter (modified in place).

The ball indices start at index 0.

Parameters
joystick:IJoystick — A reference to the IJoystick to get the ball value from
 
ball:int — A zero-based index selecting the ball to get
 
displacement:Point — Point instance for storing displacement (dx, dy)

Returns
Boolean — true on success or false on failure
