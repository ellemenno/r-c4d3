FpsMeter()Constructor
public function FpsMeter()

Create a new FpsMeter. Default target framerate of 30fps and interval meters averaging 1, 5, and 9 second frame counts will be created. These can be changed with targetRate and setFpsInterval. Monitoring starts when you call startMonitoring()

Examples
The following code shows a simple fps meter instantiation: see the constructor documentation for more options: 
package {
   import com.pixeldroid.tools.framerate.FpsMeter;
   import flash.display.Sprite;

   public class MyFpsMeterExample extends Sprite {
      public function MyFpsMeterExample() {
         super();
         var fps:FpsMeter = new FpsMeter();
         addChild(fps);
         fps.targetRate = 30;
      }
   }
}
